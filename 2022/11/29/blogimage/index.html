前言

本文将介绍，如何使用 Github 搭建图床，使用 jsDelivr 的 CDN 服务加速，通过 PicGo 上传到图床
由于网络问题，有时无法加载，建议使用国内对象存储搭建

Github图床搭建

  1. 登录 Github，新建一个仓库

    仓库设置如下：

  2. 获取 token（令牌），用于 PicGo 配置
    【Settings】→【Developer settings】→【Personal access tokens】

    找到 “Developer settings”，点击 “Personal access tokens”，生成一个 token

    token 设置：
    设置完成后拉到最下边创建即可

    注意：只会出现一次，一定要复制！一定要复制！一定要复制！
