前言

此文章为 Java核心技术卷1摘录的一些笔记，适合巩固基础，初学的朋友结合情况阅读

一些术语

  • JDK
    Java Development Kit（Java开发工具包）
    编写Java程序使用
  • JRE
    Java Runtime Environment（Java运行时环境）
    只包含虚拟机，为不需要编译器的用户提供

基本程序设计结构

基本数据类型

Java是一种强类型语言，每一个变量都需要声明类型

整型

类型 存储需求 取值范围
int 4 字节 -2^31至2^31 - 1（二十多亿）
short 2 字节 -32768（-2^15）至 32767（2^15 - 1）
long 8 字节 -2^63 至 2^63 -1
byte 1字节 -128（-2^7） 至 127（2^7 - 1）

进制表示

  • 十六进制：0x 或 0X
  • 八进制：0（容易混淆，最好不要使用）
  • 二进制：0b 或 0B（Java7开始）

浮点型

类型 存储需求 取值范围
float 4 字节 精度6~7位
double 8 字节 精度15位

没有 F 或 f 后缀的浮点值默认为 double

浮点数值存在误差：
因为浮点数值采用二进制表示，而在二进制中无法精确表示 1/10。相当于十进制无法精确表示 1/3 一样

char类型

char 类型表示单个字符或 Unicode 字符

转义序列

名称 转义序列 Unicode值
退格 \b \u0008
制表 \t \u0009
换行 \n \u000a
回车 \r \u000b
双引号 \“ \u0022
单引号 \‘ \u0027
反斜杠 \\ \u005c

注意：Unicode 转义序列会在解析代码前处理

boolean类型

只有false 和 true 两个值

数据类型的转换

小转大自动转换，整型转浮点型可能有精度损失（整数位数超出浮点型最大位数）
大转小强制类型转换，可能丢失信息

对数值强制转换时，若超出表示范围，会截断成完全不同的值

变量

变量名以字母开头，不能使用保留字作变量名
不能使用未初始化的变量
变量声明最好尽可能靠近第一次使用的地方

从 Java10 开始，对于局部变量，如果能从变量的初始值推断出它的类型，可以使用 var 关键字而无需指定类型

常量

使用 final 修饰的变量一旦被赋值，不能再修改
常量名全大写
例如：final PI = 3.14;

运算符

算术运算符

除法时，若两数都为整数，表示整数除法，否则表示浮点除法

关系和逻辑运算符

返回结果为 boolean 类型

逻辑运算符存在短路现象，&& 或 || 第一个条件符合时，第二个条件不进行判断

Math数学函数

  • sqrt 平方根
  • pow 幂运算
    pow(x, a) –> x 的 a 次幂
  • round 返回浮点数最接近的整数
  • random 随机浮点数 [0,1)

字符串

  • substring 取子串
  • concat 或 + 拼接字符串
  • equals 检测两个字符串是否相等
  • StringBuilder 类

输入

使用 Scanner 类读取输入，该类定义在 java.util 包中，需要导入

循环

循环条件为：检测两个浮点数是否相等时要注意，由于 0.1 无法精确地用二进制表示，可能会死循环

case 标签可以是：

  • char、byte、short 或 int 类型的常量表达式
  • 枚举常量
  • 字符串字面量（Java7开始）

大数

BigInteger 实现任意精度的整数计算
BigDecimal 实现任意精度的浮点数计算

大数不能使用算数运算符计算，需要 add 等相关方法进行运算

数组

for each 遍历数组输出

  • Arrays.copyOf 拷贝
  • Arrays.sort 排序

对象与类

对象与对象变量

对象变量并没有实际包含一个对象，而是引用一个对象

String s = new String();
上边的表达式 new String() 构造了一个字符串对象，它的值是对新创建的对象的一个引用。这个引用存储在 s 中。

LocalDate 类

不要使用构造器来构造 LocalDate 类的对象。实际上，应当使用静态工厂方法，它会代表你调用构造器

  • LocalDate.now()
  • LocalDate.of(year, month, day)
